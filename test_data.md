  로컬 설치형 (--path 스캔 병행 가능)

  ┌───────────────┬───────────────────────────────────────────┐
  │     도구      │                   특징                    │
  ├───────────────┼───────────────────────────────────────────┤
  │ OWASP WebGoat │ 의도적 취약 Java 앱. 소스코드 스캔도 가능 │
  ├───────────────┼───────────────────────────────────────────┤
  │ DVWA          │ PHP 취약 앱. Docker로 빠르게 실행 가능    │
  └───────────────┴───────────────────────────────────────────┘

  # DVWA Docker 실행 후 URL 스캔
  docker run -d -p 80:80 vulnerables/web-dvwa
  python scan.py --url http://localhost --skip-zap

  ---
  추천 테스트 순서

  # 1. 빠른 검증 (ZAP 제외, nmap은 Docker 필요)
  python scan.py --url http://testphp.vulnweb.com --skip-zap

  # 2. 이 프로젝트 기존 기준치 비교
  python scan.py --url http://testfire.net --skip-zap

  # 3. OWASP Top 10 전체 커버리지 확인
  python scan.py --url https://juice-shop.herokuapp.com --skip-zap

  인스턴스에서만 사용하세요.
